<- Room -------------------------------------------------------------------------------------------

Класс, описывающий комнату, в которой будет проходить игровая сессия

>>>>>>>>>>>>>>>>>>>>>>>ПОЛЯ<<<<<<<<<<<<<<<<<<<<<<<

    private $_id - идентификатор комнаты. Выдается конструктором
    private $_users - массив пользователей (src/User.doc.txt)
    private $_game - игровая сессия комнаты (src/Game.doc.txt)

>>>>>>>>>>>>>>>>>>>>>>МЕТОДЫ<<<<<<<<<<<<<<<<<<<<<<

public function addUser(User &$new_user) - добавить пользователя. Вызывает сокет-сервер

public function isFull() - Проверяет, есть ли место в комнате

public function getUsersNames() - возвращает массив из 4 логинов текущих пользователей. Если
    пользователей меньше 4х, некоторые значения массива будут равны null

public function draw() - функция для дебага (не используется)

public function getUsersSockets() - возвращает массив из 4 сокетов текущих пользователей. Если
    пользователей меньше 4х, некоторые значения массива будут равны null

public function getUsersIPs() - возвращает массив из 4 IP-адресов текущих пользователей. Если
    пользователей меньше 4х, некоторые значения массива будут равны null

public function getUsersPts() - возвращает массив из 4 рангов текущих пользователей (новые
    пользователи получают ранг 500 при регистрации на сайте). Если пользователей меньше 4х,
    некоторые значения массива будут равны null

public function getUsersCount() - возвращает кол-во подключенных пользователей

public function sendMsgToChat($msg_owner, $msg_content) - отсылает сообщение в чат для
    всех пользователей данной комнаты
